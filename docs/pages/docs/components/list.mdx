Lists consist of one or more list items, and can contain actions represented by
icons and text. List items come in three sizes: one-line, two-line, and
three-line.

- [Design article](https://m3.material.io/components/lists)
- API Documentation (coming soon)
- [Source code](https://github.com/efoken/md3-ui/tree/main/packages/list)

## Usage

```jsx
<List>
  <ListItem headline="Headline" supportingText="Supporting text" />
</List>
```
